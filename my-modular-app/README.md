# Модульная архитектура React приложения

## Что такое модульная архитектура?
Это подход, при котором код разделён на независимые модули. Каждый модуль отвечает за свою функциональность (пользователи, товары, посты) и имеет свои компоненты.

## Структура проекта
src/
  modules/
    common/        # общие компоненты (кнопка, шапка)
    user/          # модуль пользователя
      components/  # внутренние компоненты модуля
      UserCard.js  # публичный экспорт
    posts/         # модуль постов

## Почему это удобно?
- Легко найти нужный код
- Модули можно переиспользовать в других проектах
- Легко добавлять новые фичи, не ломая старые

## Как запустить
npm install
npm run dev
